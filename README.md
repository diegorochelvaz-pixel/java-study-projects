Industrial Machine Simulation - Java
Objective
Develop a Java application to simulate the operation of industrial machines using Object-Oriented Programming concepts and multithreading.
The project represents machines commonly found in manufacturing environments such as CNC lathes, milling machines, and air compressors.

Technologies Used
 - Java
 - OOP (Object-Oriented Programming)
 - Inheritance
 - Interface
 - Polymorphism
 - Method Override
 - Threads
 - ExecutorService
 - Runnable
 - Implemented Classes
 - Machine (Base Class)

Responsible for storing common machine parameters:
 - name
 - temperature
 - pressure

Also implements:
 - equipment startup and shutdown
 - Runnable interface for thread execution
 - CNC Lathe

Simulates shaft machining operations.
Milling Machine
Simulates surface milling operations.
Compressor
Simulates compressed air generation.
Practical Application

This project was developed with an industrial mindset, connecting programming concepts to real manufacturing equipment and process simulation.
